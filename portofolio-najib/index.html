// 01 — Tentang

Halo, Saya Najib

Nama saya Najib Izyan Fahrezi, mahasiswa aktif semester 2 Program Studi Informatika di Universitas Ahmad Dahlan (UAD), Yogyakarta. Saya adalah seseorang yang percaya bahwa teknologi dan olahraga bisa berjalan beriringan — keduanya membutuhkan dedikasi, strategi, dan kerja keras yang konsisten.

Di bangku kuliah, saya sedang mendalami dunia pemrograman, desain web, dan logika komputasi. Saya menikmati proses belajar hal-hal baru, terutama ketika saya bisa langsung mempraktikkannya dalam proyek nyata. Setiap baris kode bagi saya seperti taktik di lapangan — harus tepat, terencana, dan bekerja sama dengan bagian lainnya.

Di luar dunia akademik, saya aktif sebagai pemain sepak bola di tim UAD. Olahraga ini mengajarkan saya nilai disiplin, komunikasi, dan semangat pantang menyerah yang sangat berguna dalam kehidupan sehari-hari, termasuk saat menghadapi tantangan perkuliahan.
